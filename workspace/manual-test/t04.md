# T04 — Manual/Live UAT: stakeholder role enum + stage-in-pipeline FK

Prereqs: `make infra-up && make migrate-up && make seed-reset && make run` (or the
project's standard live-stack bootstrap).

1. **[auto]** Run `make migrate-status`.
   Expected: current version is `63`, not dirty.

2. **[auto]** Run `make migrate-down` then `make migrate-up`.
   Expected: both exit 0; `make migrate-status` again reports version `63`, not dirty
   (clean round-trip).

3. **[live]** Connect to the dev DB (`make psql`) and run:
   ```sql
   INSERT INTO relationship (workspace_id, kind, person_id, deal_id, source, captured_by)
   VALUES ('00000000-0000-0000-0000-000000000001', 'deal_stakeholder',
           '00000000-0000-0000-0001-000000000001',
           '00000000-0000-0000-0042-000000000001', 'manual', 'human:uat');
   ```
   Expected: rejected with a CHECK constraint violation (`rel_stakeholder_role_enum`) —
   role NULL is disallowed for `kind='deal_stakeholder'`.

4. **[live]** Re-run the same insert with `role` set to `'champion'` added to the
   column/value lists.
   Expected: succeeds (1 row inserted).

5. **[live]** Run the same insert again with `role` set to `'champion'` but for an
   `employment`-kind row instead (swap `kind`/`deal_id` for `kind='employment'` +
   `organization_id`, and set `role` to `NULL`, using a real person/org id pair).
   Expected: succeeds — the CHECK is scoped to `deal_stakeholder`, not table-wide.

6. **[auto]** Run:
   ```bash
   make test-it DIR=backend/internal/modules/directory RUN='TestRelationshipStakeholderRoleEnum|TestDealCompositeFK'
   ```
   Expected: both `PASS` — proves DEAL-EXT-5 (role enum) and reconfirms DEAL-EXT-6
   (deal's stage must belong to its own pipeline, DB-guaranteed composite FK) together.
   (`make test-it` provisions `TEST_DATABASE_URL` against a throwaway DB clone; running the
   bare `go test -tags integration ...` command directly will fail with `TEST_DATABASE_URL
   not set` since that env var is only exported by the Make target, not the shell.)

7. **[auto]** Run `make check`.
   Expected: exits 0.
