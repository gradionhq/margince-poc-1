# T12 Live UAT Guide

Run this only after the stack is up with `infra-up -> migrate-up -> seed-reset -> run`.
The commands below assume:

- the API is listening on `http://localhost:8080`
- the database is reachable through `DATABASE_URL`
- you can act as a tenant by sending `X-Workspace-Id` + `X-User-ID`
- Python 3 is available (used to mint `X-Approval-Token` values in UAT-3 and UAT-5)

Use one shell session so the variables set in the bootstrap block stay available for the later steps.

## Prerequisites: `APPROVAL_TOKEN_SIGNING_SECRET`

The approval-token seam reads this env var at runtime. It must be set **both** in the
shell where you run `make run` (so the server has it) and in the shell where you run the
UAT steps below (so the token-minting snippets match). Add it to your `.env.local` before
starting the server:

```bash
# in .env.local (or export before make run)
APPROVAL_TOKEN_SIGNING_SECRET=dev-t12-uat-secret
```

Then in this UAT shell:

```bash
export APPROVAL_TOKEN_SIGNING_SECRET=dev-t12-uat-secret
```

The value must match between the server and the shell. Any non-empty string works for dev;
the `dev-t12-uat-secret` value above is the convention for this guide.

## Step 1: Create an agent passport

Agent calls must use `Authorization: Bearer <token>`. Mint a passport for the UAT user so
the middleware sets `IsAgent: true` on those requests:

```bash
export AGENT_TOKEN=$(curl -sS -X POST \
  -H "Content-Type: application/json" \
  -H "X-Workspace-Id: ${WS_ID}" \
  -H "X-User-ID: ${USER_ID}" \
  -d '{"scopes":[],"expires_in_seconds":3600}' \
  "${API_BASE}/passports" | python3 -c "import sys,json; print(json.load(sys.stdin)['token'])")
echo "AGENT_TOKEN=${AGENT_TOKEN}"
```

Expected: `POST /passports` returns `201 Created` and `AGENT_TOKEN` is a non-empty string.

## UAT-1: Open → open — agent caller, no token needed (🟢)

```bash
# Create deal in Qualify (open-a).
export DEAL1=$(curl -sS -X POST \
  -H "Content-Type: application/json" \
  -H "X-Workspace-Id: ${WS_ID}" \
  -H "X-User-ID: ${USER_ID}" \
  -d "{\"name\":\"T12 UAT-1 deal\",\"pipeline_id\":\"${PIPELINE}\",\"stage_id\":\"${STAGE_OPEN_A}\",\"amount_minor\":5000,\"currency\":\"USD\",\"source\":\"uat\",\"captured_by\":\"human:t12\"}" \
  "${API_BASE}/deals")
export DEAL1_ID=$(echo "$DEAL1" | python3 -c "import sys,json; print(json.load(sys.stdin)['id'])")
echo "DEAL1_ID=${DEAL1_ID}"

# Advance as agent (open → open) — no X-Approval-Token needed for a 🟢 transition.
curl -i -sS -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AGENT_TOKEN}" \
  -d "{\"to_stage_id\":\"${STAGE_OPEN_B}\"}" \
  "${API_BASE}/deals/${DEAL1_ID}/advance"
```

Expected: `200 OK`. Response body has `status: "open"` and `closed_at: null`.

Verify exactly one write per table:

```bash
psql "$DATABASE_URL" -c "SELECT count(*) AS history_rows FROM deal_stage_history WHERE deal_id='${DEAL1_ID}' AND to_stage_id='${STAGE_OPEN_B}';"
psql "$DATABASE_URL" -c "SELECT count(*) AS audit_rows  FROM audit_log WHERE entity_id='${DEAL1_ID}' AND action='advance_stage';"
psql "$DATABASE_URL" -c "SELECT count(*) AS event_rows  FROM event_outbox WHERE entity_id='${DEAL1_ID}' AND topic='deal.stage_changed';"
```

Expected: all three counts are `1`.

## UAT-3: Agent with valid token succeeds; replay rejected

```bash
# Reuse DEAL2_ID (still in open-a from UAT-2).

# Mint an X-Approval-Token for this exact advance (deal, target stage, status).
export APPR_TOKEN=$(python3 - "${DEAL2_ID}" "${STAGE_WON}" "won" "uat3-jti-${DEAL2_ID}" <<'PYEOF'
import hmac as _hmac, hashlib, base64, json, os, sys
from datetime import datetime, timezone, timedelta

def b64u(b):
    return base64.urlsafe_b64encode(b).rstrip(b'=').decode()

root = os.environ['APPROVAL_TOKEN_SIGNING_SECRET']
ws_id = os.environ['WS_ID']
deal_id, to_stage_id, status, jti = sys.argv[1], sys.argv[2], sys.argv[3], sys.argv[4]

# Per-workspace key: HMAC-SHA256(root, workspace_id)
key = _hmac.new(root.encode(), ws_id.encode(), hashlib.sha256).digest()

# diff_hash: SHA-256 of the canonical (sort_keys) JSON of the bound fields
diff = {'deal_id': deal_id, 'to_stage_id': to_stage_id, 'status': status}
diff_hash = b64u(hashlib.sha256(
    json.dumps(diff, sort_keys=True, separators=(',', ':')).encode()
).digest())

exp = (datetime.now(timezone.utc) + timedelta(minutes=5)).strftime('%Y-%m-%dT%H:%M:%SZ')
claims = {'jti': jti, 'approval_id': 'uat-' + jti, 'workspace_id': ws_id,
          'tool': 'advance_deal', 'diff_hash': diff_hash, 'exp': exp, 'single_use': True}

hdr = b64u(json.dumps({'alg': 'HS256', 'typ': 'JWT'}, separators=(',', ':')).encode())
pay = b64u(json.dumps(claims, separators=(',', ':')).encode())
sig = _hmac.new(key, (hdr + '.' + pay).encode(), hashlib.sha256).digest()
print(hdr + '.' + pay + '.' + b64u(sig))
PYEOF
)
echo "APPR_TOKEN=${APPR_TOKEN}"

# First use: agent advances with valid token.
curl -i -sS -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AGENT_TOKEN}" \
  -H "X-Approval-Token: ${APPR_TOKEN}" \
  -d "{\"to_stage_id\":\"${STAGE_WON}\",\"status\":\"won\"}" \
  "${API_BASE}/deals/${DEAL2_ID}/advance"
```

Expected: `200 OK`. Response body has `status: "won"`, non-null `closed_at`, and non-null
`fx_rate_to_base` (frozen from the seeded USD→EUR rate of 0.93).

Replay the same token:

```bash
curl -i -sS -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AGENT_TOKEN}" \
  -H "X-Approval-Token: ${APPR_TOKEN}" \
  -d "{\"to_stage_id\":\"${STAGE_WON}\",\"status\":\"won\"}" \
  "${API_BASE}/deals/${DEAL2_ID}/advance"
```

Expected: `403 Forbidden` with body `{"code":"approval_token_invalid",...}`.

```bash
export DEAL5=$(curl -sS -X POST \

export DEAL5_ID=$(echo "$DEAL5" | python3 -c "import sys,json; print(json.load(sys.stdin)['id'])")

# Human closes to lost (no token required for humans on 🟡 transitions).
curl -i -sS -X POST \
  -H "Content-Type: application/json" \
  -H "X-Workspace-Id: ${WS_ID}" \
  -H "X-User-ID: ${USER_ID}" \
  -d "{\"to_stage_id\":\"${STAGE_LOST}\",\"lost_reason\":\"budget cut\"}" \

```

Expected: `200 OK` with non-null `closed_at`, `lost_reason: "budget cut"`, and non-null
`fx_rate_to_base`.

Agent attempts to reopen without token:

```bash
curl -i -sS -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AGENT_TOKEN}" \
  -d "{\"to_stage_id\":\"${STAGE_OPEN_A}\"}" \

```

Expected: `403 Forbidden` with body `{"code":"approval_required",...}`.

Mint a reopen token (lost → open is 🟡 because the from-stage is terminal):

```bash
export REOPEN_TOKEN=$(python3 - "${DEAL5_ID}" "${STAGE_OPEN_A}" "open" "uat5-reopen-${DEAL5_ID}" <<'PYEOF'
import hmac as _hmac, hashlib, base64, json, os, sys
from datetime import datetime, timezone, timedelta

def b64u(b):
    return base64.urlsafe_b64encode(b).rstrip(b'=').decode()

key = _hmac.new(root.encode(), ws_id.encode(), hashlib.sha256).digest()

diff = {'deal_id': deal_id, 'to_stage_id': to_stage_id, 'status': status}

    json.dumps(diff, sort_keys=True, separators=(',', ':')).encode()
).digest())

exp = (datetime.now(timezone.utc) + timedelta(minutes=5)).strftime('%Y-%m-%dT%H:%M:%SZ')
claims = {'jti': jti, 'approval_id': 'uat-' + jti, 'workspace_id': ws_id,
          'tool': 'advance_deal', 'diff_hash': diff_hash, 'exp': exp, 'single_use': True}

hdr = b64u(json.dumps({'alg': 'HS256', 'typ': 'JWT'}, separators=(',', ':')).encode())
pay = b64u(json.dumps(claims, separators=(',', ':')).encode())
sig = _hmac.new(key, (hdr + '.' + pay).encode(), hashlib.sha256).digest()
print(hdr + '.' + pay + '.' + b64u(sig))

)

# Agent reopens with valid token.
curl -i -sS -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${AGENT_TOKEN}" \
  -H "X-Approval-Token: ${REOPEN_TOKEN}" \
  -d "{\"to_stage_id\":\"${STAGE_OPEN_A}\"}" \
  "${API_BASE}/deals/${DEAL5_ID}/advance"
```

Expected: `200 OK`. Response body has `status: "open"`, `closed_at: null`,
`lost_reason: null`, and `fx_rate_to_base: null` — all cleared by the reopen path.
