## UAT Item 8: Live API [live]

Boot the full stack:

```bash
make infra-up && make migrate-up && make seed-reset && make run
```

Seed an open deal with `last_activity_at` older than 60 days (or reuse the seed data):

```bash
# Patch a deal to be stale (replace <DEAL_ID> and <WS_ID> with real values from seed output)
psql "$TEST_DATABASE_URL" -c \
  "UPDATE deal SET last_activity_at = now() - interval '70 days' WHERE id='<DEAL_ID>'::uuid"
```

Query the `listDeals` endpoint with `stalled=true`:

```bash
# The API server listens on port 8080 (crm.yaml servers[0]: http://localhost:8080/v1)
curl -s "http://localhost:8080/v1/deals?stalled=true" \
  -H "X-Workspace-ID: <WS_ID>" | jq '.data[].id'
```

**Expected:** The stale deal's `id` appears in `data[]`. A fresh deal (last_activity_at within 60 days) must NOT appear. A deal with an active `wait_until` (future date) must NOT appear even if idle >60 days.
